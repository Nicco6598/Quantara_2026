# UI Parity Notes

- Sidebar remains persistent and includes active project context.
- Topbar includes date range, filters, export and primary SAL CTA.
- Dashboard starts with dense KPI cards and project health.
- Central workspace uses dense tabular project data.

- Alert list mirrors the legacy "Alert attivi" panel with severity badges.

- Map card restores the construction route pattern with live status waypoints.
- Timeline card restores the Gantt-style project planning view.
- Operations strip restores site/material/task counters from the dashboard baseline.
