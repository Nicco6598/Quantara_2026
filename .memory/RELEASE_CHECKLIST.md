# Release Checklist

## Required before first distributable

- [ ] SemVer version reviewed.
- [ ] `CHANGELOG.md` updated.
- [ ] Rust `cargo check` green.
- [ ] Biome format and lint green.
- [ ] TypeScript strict typecheck green.
- [ ] Unit and integration tests green.
- [ ] Playwright smoke green.
- [ ] SQLite migration test green.
- [ ] Backup and restore path tested.
- [ ] Updater configuration finalized.
- [ ] Patch notes UI verified.
- [ ] Signing process documented.
